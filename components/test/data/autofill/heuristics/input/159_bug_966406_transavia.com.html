Où puis-je trouver mon numéro de réservation?

Si vous avez effectué votre réservation sur notre site internet ou via notre Service Clients, vous trouverez votre numéro de réservation sur la confirmation de réservation reçue par e-mail. Si vous avez réservé votre vol depuis un tour opérateur ou un agent de voyage, vous trouverez votre numéro de réservation sur les documents de voyage transmis par votre tour opérateur ou votre agent de voyage.

# How to Contribute

1. The main branch is [/mtp](https://github.com/Bivas-Biswas/malabr/tree/mtp)

2. Create a separate features branch

    a. Sync the latest update `git pull --rebase origin mtp`

    b. create a branch `git checkout -b your_name/features1`
    
    c. push of your change to

3. Raise a pull request to `mtp` branch with proper title

4. Ask for a review

5. Do `squash and merge` commit
